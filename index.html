Apa itu Ruang Coding?

Ruang Coding adalah sebuah platform untuk belajar programming yang didirikan oleh 5 orang dengan tujuan untuk membagi ilmu dengan cara yang mudah dipahami dan mengasyikan.

Services

Menjadi Front-End Developer

Di kursus ini, kalian akan belajar tentang Front-End Development dari dasar hingga mahir. Kalian akan belajar tentang struktur website dengan HTML, memperindah tampilan website kalian dengan CSS dan membuat web kalian menjadi interaktif dengan JavaScript. Di akhir kursus ini, kalian akan bisa membuat sebuah website milik kalian sendiri.

Menjadi Back-End Developer

Di kursus ini, kalian akan belajar tentang Back-End Development dari dasar hingga mahir. Kalian akan belajar tentang server, routing di Node JS, mengelolah database menggunakan MYSQL dan membuat web kalian dinamis. Di akhir kursus ini, kalian akan membuat sebuah website e-commerce.

Menjadi Android Developer

Di kursus ini, kalian akan belajar tentang Android Development dari dasar hingga mahir. Kalian akan belajar menggunakan software Android Studio, dan kalian juga akan belajar bahasa Kotlin untuk membuat sebuah aplikasi Android. Di akhir kursus ini, kalian akan bisa membuat sebuah aplikasi pemesanan tiket milik kalian.
